HTML Structure


The html template uses Latest Bootstrap V5.2.1 with valid HTML5 tags. This theme has a responisve layout with 4 column support. All of the information in content area is nested within a class and comes with predefined classes.

Basic Grid HTML

For a simple two column layout, create a .container and add the appropriate number of .col-md* columns.

Given this example, we have .col-md-4 and .col-md-8, making for 12 total columns and a complete row.

Nesting Columns

To nest your content with the default grid, add a new .container and set of .col-md* columns within an existing .col-md* column. Nested rows should include a set of columns that add up to the number of columns of its parent.

Jquery and javascript


This theme imports three types of Javascript files.

  • 1. jQuery
  • 2. Custom scripts (main.js)
  • 3. Some jQuery Based Plugins

All the js included under

./assets
  |- /js
    |- /...
    |- /...
    |- /...
    |- /vendor

The vendor folder includes all thrid-party scripts like jQuery and jQuery Based Plugins

The version of jQuery included is

jQuery version v3.6

All of the plugin initailization and implementation is included inside 

./assets
  |- /js
    |- /plugin.js

You can change any of the plugin behavior or functionality from the file.



These are the JS files that are loaded into templates in end of the Body Section.
